# Security Policy

## Supported Versions

As this is an issue tracking repository for the Bemanning application, we track security issues related to the latest deployed version of the application.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

We take the security of Bemanning seriously. If you have found a security vulnerability in the Bemanning application, please report it to us as soon as possible.

### How to Report

Please do **NOT** open a public issue for security vulnerabilities. Instead, please report them privately:

1. Email us at ignasi.fernandez@chalmers.se
2. Provide a detailed description of the vulnerability
3. Include steps to reproduce the issue

### What to Expect

1. We will acknowledge your report within 48 hours.
2. We will investigate the issue and determine its impact.
3. We will keep you updated on our progress.
4. Once resolved, we will notify you and may credit you for the discovery (unless you prefer to remain anonymous).

Thank you for helping keep Bemanning and its users safe!
